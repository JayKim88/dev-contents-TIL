# 배열과 포인터2

## 포인터의 개념

- 포인터(pointer) : 변수의 일종
  - 변수: 특정 데이터 값을 가지고 있음
  - 포인터: 특정 데이터가 저장된 기억장소의 주소(번지)값을 가지고 있음
- 따라서 포인터는 기억공간을 변수명으로 접근하지 않고 주소로 접근하기 위해 사용
